# Question App

https://nihangonenc-question-app.netlify.app/
Question App toplam 10 sorudan oluşmaktadır. Her bir sorunun 30 saniye süresi vardır. Sorunun seçenekleri 10 saniye sonra yüklenmektedir.

## Kullanılan Teknolojiler

- React.js
- JavaScript
- Vite
- HTML
- CSS
